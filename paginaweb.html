StarTrek

Proyecto

Información

Este proyecto consiste en unir tres materias de la UPP donde se pretende llevar a cabo un sitio web que permita tener organizada la informacion perteneciente a StarTrek, la cual posee un login, registro y una aplicación de juegos.

Proyecto

BASE DE DATOS

Objetivo

El objetivo del presente proyecto es la elaboración de una base de datos funcional que debe cumplir una serie de requerimientos a partir de los conocimientos adquiridos en la materia de base de datos los cuales permiten obtener: Independencia lógia y física de los datos.

LABORATORIO DE PROGRAMACIÓN

Objetivo

El objetivo de este proyecto es aprender a dominar el desarrollo de una pagina web dinámica e interactiva con las funciones mas importantes, como: login, registro, conectar una base de datos y validaciones de usuarios.

LABORATORIO DE APLICACIONES

Objetivo

El objetivo es desarrollar aplicaciones utilizando el motor de App Inventor que utiliza el sistema de bloques con funciones preestablecidas para la programación de juegos, apps, etc.

Aplicación

StarTrek Games

Cuenta con dos mini juegos mentales relacionados a StarTrek y un visualizador a la pagina web
